# DXOpal

Game development framework for [Opal](http://opalrb.org/),
API compatible to [DXRuby](http://dxruby.osdn.jp/)

## Demo

https://yhara.github.io/dxopal/

## Status

POC (just started)

## How to run demo

1. git clone
1. bundle install
1. bundle exec rackup
