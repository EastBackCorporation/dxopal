API difference

Loading resources

Because of the async nature of JavaScript, Image.load and Sound.new are changed to Image.register, Sound.register and Window.load_resources.

Window.load_resources tries to load all the resouces declared by register and call the block when all of them are loaded.

Image.register(:player, 'images/player.png')
Sound.register(:game_over, 'sounds/game_over.wav')

Window.load_resources do
  Window.loop do
    # ... Window.draw(0, 0, Image[:player]) ...

    # ... Sound[:game_over].play ...
  end
end
